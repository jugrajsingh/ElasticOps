Features

Everything your cluster needs

📊
Cluster Overview

Real-time health, node stats, JVM memory, disk usage, and shard counts — all in one glanceable dashboard.

🞕
Shard Map

Visual shard-level layout across nodes. Spot imbalances, unassigned shards, and relocation events instantly.

🔍
Index Analysis

Per-index optimization recommendations: segment counts, merge candidates, replica ratios, and storage efficiency scores.

⚙
Job Engine

Async job execution for force-merge, shard reduction, and index operations — with full audit history and progress tracking.

🌐
Multi-Cluster

Connect and switch between multiple Elasticsearch clusters from a single UI. No config files to hand-edit.

🔒
Auth & RBAC

Local JWT authentication with bcrypt hashing. Per-user cluster access controls with an admin panel to manage users.
